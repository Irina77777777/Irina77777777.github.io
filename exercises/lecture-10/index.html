CSS Flex Property

CSS
Flex
Property
- розподілити простір між елементами flex так, щоб вони мали однакову ширину та заповнювали весь доступний простір. Закоментуйте цей рядок. - Встановіть для елемента blue фактор зростання 3 - Встановіть для елемента green фактор зростання 1 - Розрахуйте в пікселях ширину кожного з елементів item-green, item-orange, item-blue

item-green 700 / 6 * 2 = 233px;

item-orange 700 / 6 = 117px;

item-blue 700 /6 * 3 = 350px;
